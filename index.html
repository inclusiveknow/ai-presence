what am i looking at?

A live map of the artificial intelligence already running on Earth.

The dim points are cumulative known systems — neural networks, large models, deployed AI services — placed at the labs, datacenters and hubs that built or host them. The map fills as the timeline advances from 1999 to today.

The pulses are live activity, polled in real time:

  • models recent uploads to Hugging Face's public model hub
  • code public GitHub events on AI-tagged repositories
  • inference heartbeat at major datacenter regions, weighted by time-of-day usage

Sources: Hugging Face public model API, GitHub public events API, Epoch AI's notable ML models database, arXiv cs.LG counts. Historical year markers are calibrated against published research curves.

No tracking. No analytics. No personal data. The browser fetches public APIs directly — view source if you want to confirm.
